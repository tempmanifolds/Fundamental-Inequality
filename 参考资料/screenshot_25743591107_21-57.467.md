# 基本不等式的教学思路

## 发现 → 证明 → 应用

### 求最值问题前，首先要理解基本不等式中"等号成立"的意义

**问题 3-1**　如图，当不等式 $\dfrac{a+b}{2} \geq \sqrt{ab}$ 等号成立时，此时 $D$ 点的位置在哪？

> 【可能答案，待核】等号成立 $\Leftrightarrow a = b$ $\Leftrightarrow$ $C$ 为 $AB$ 中点 $\Leftrightarrow$ $D$ 在过 $C$ 的垂直方向上（即 $D$ 与 $E$ 重合，$CD$ 取最大值）。

**问题 3-2**　线段 $DC$ 与 $DO$ 之间的长度满足什么关系？

> 【可能答案，待核】$DC \leq DO$，即 $\sqrt{ab} \leq \dfrac{a+b}{2}$。

**问题 3-3**　线段 $DC$ 什么时候长度最大，为什么？

> 【可能答案，待核】当 $C$ 移到圆心 $O$（即 $a = b$）时，$DC$ 取最大值为半径 $\dfrac{a+b}{2}$。

**问题 3-4**　当线段 $DC$ 长度最大时，$D$ 点在什么位置，此时不等式的等号成立吗？

> 【可能答案，待核】此时 $D$ 位于过 $C$ 的垂直方向（$D$ 与 $E$ 重合），$a = b$，不等式 $\dfrac{a+b}{2} \geq \sqrt{ab}$ 的等号成立。

> 【图示】圆 $O$ 中，$AB$ 为直径，$C$ 在 $AB$ 上，$AC = a$，$BC = b$，$DE$ 过 $C$ 且 $DE \perp AB$，$D$ 在上半圆，$E$ 在下半圆，连接 $AD$、$BD$ 构成直角三角形。
