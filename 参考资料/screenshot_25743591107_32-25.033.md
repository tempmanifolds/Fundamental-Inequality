# 进一步认识基本不等式——函数视角

## 平均数定义

令

$$A = \dfrac{a+b}{2}, \quad H = \dfrac{2ab}{a+b}, \quad G = \sqrt{ab}, \quad Q = \sqrt{\dfrac{a^2 + b^2}{2}}$$

## 构造统一函数

$$f(x) = \dfrac{b - x}{x - a} = -1 + \dfrac{b - a}{x - a}$$

代入得：

$$f(A) = 1, \quad f(H) = \dfrac{b}{a}, \quad f(G) = \sqrt{\dfrac{b}{a}}, \quad f(Q) = \dfrac{Q + a}{b + Q}.$$

## 单调性证明

不妨设 $a \leq b$，则 $f(x)$ 在 $(a, +\infty)$ 单调递减。

因为

$$\dfrac{b}{a} \geq \sqrt{\dfrac{b}{a}} \geq 1 \geq \dfrac{Q + b}{b + Q} \geq \dfrac{Q + a}{b + Q},$$

且 $A, H, G, Q \in (a, +\infty)$，所以

$$f(H) \leq f(G) \leq f(A) \leq f(Q),$$

即

$$\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leq \sqrt{ab} \leq \dfrac{a+b}{2} \leq \sqrt{\dfrac{a^2 + b^2}{2}}.$$

> 【注】原图中链式不等式按 $\dfrac{b}{a} \geq \sqrt{b/a} \geq 1 \geq \dfrac{Q+b}{b+Q} \geq \dfrac{Q+a}{b+Q}$ 排列，并由 $f$ 单调递减得到 $f(H) \leq f(G) \leq f(A) \leq f(Q)$，即平均数链。
