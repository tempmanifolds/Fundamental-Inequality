# 进一步认识基本不等式——应用视角

## "知一求三"

平均数链：

$$\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leq \sqrt{ab} \leq \dfrac{a+b}{2} \leq \sqrt{\dfrac{a^2 + b^2}{2}}$$

若 $\sqrt{ab} = 1$，则

$$\dfrac{a+b}{2} \geq 1, \quad \dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leq 1, \quad \sqrt{\dfrac{a^2 + b^2}{2}} \geq 1$$

## 例题与变式

**问题 1**　已知正数 $a, b$，且 $ab = 1$，则 $a + b$ 的最小值为 ______。

> 【答案】$2$。

**变 1**　$(a+1)(2b+1) = 1$，则 $a + 2b + 2$ 的最小值为 ______。

**变 2**　$2ab + a + 2b = 0$，则 $a + 2b$ 的最小值为 ______。

> 【注】原图未给具体答案。

**变 3**　$2ab + a + 2b = 2$，则 $2a + 2b$ 的最小值为 ______。

> 【提示】原图给出推导路径：
>
> $$(a+1)(2b+1) = 3 \;\cdots\; 2(a+1)(2b+1) = (2a+2)(2b+1) = 6$$
>
> 由基本不等式得 $2a + 2 \geq 2\sqrt{2a}$，$2b + 1 \geq 2\sqrt{2b}$，从而 $6 \geq 4\sqrt{2a \cdot 2b}$，求出 $2a+2b$ 的最小值。

**变 4**　$a^2 + 3ab + 2b^2 = 1$，则 $2a + 3b$ 的最小值为 ______。

## 解题策略

- **换元法**：通过条件变形，将问题转化为标准的"知一求三"模型。
- **凑配法**：通过乘系数、加项等手段凑出能直接套用基本不等式的形式。
