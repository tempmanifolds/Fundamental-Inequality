# 基本不等式——基本运算

## 如何更好地理解几何平均数

### 算术平均数

$$a + b = \dfrac{a+b}{2} + \dfrac{a+b}{2}$$

$$a + b + c = \dfrac{a+b}{3} + \dfrac{a+b}{3} + \dfrac{a+b}{3}$$

算术平均数针对的是**加法运算**，实质是把若干个数字的和用等数量的同一个数字的和来代替（加法平均数）。

### 几何平均数

$$ab = \sqrt{ab} \cdot \sqrt{ab}$$

$$abc = \sqrt[3]{abc} \cdot \sqrt[3]{abc} \cdot \sqrt[3]{abc}$$

几何平均数针对的是**乘法运算**，实质是把若干个数字的积用等数量的同一个数字的积来代替（乘法平均数）。
