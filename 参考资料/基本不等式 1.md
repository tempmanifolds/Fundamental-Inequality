


### 一、第9课时：基本不等式（1）

#### 【新课引入】

当 a= b 时，等号成立。

特别地，如果 a > 0，b > 0，用 √a、√b 分别代替上式中的 a、b，可得：

$$a + b \geq 2\sqrt{ab}$$

#### 【讲授新课】

##### 知识点1：基本不等式

当且仅当 a = b 时，等号成立。

$$\frac{a+b}{2} \geq \sqrt{ab}$$

- $\frac{a+b}{2}$ 叫做正数 a、b 的**算术平均数**
- $\sqrt{ab}$ 叫做正数 a、b 的**几何平均数**

**① 基本不等式表明：** 两个正数的算术平均数不小于它们的几何平均数。

**基本不等式的无字证明：** 《几何原本》卷2的几何代数法（用几何方法研究代数问题）成了后世数学家处理问题的重要依据，通过这一原理，很多代数公理、定理都能够通过图形证明，并称之为"无字证明"。

**② 基本不等式的常见变形**

成立条件及取等号条件为：

- 变形1：$ab \leq \left(\frac{a+b}{2}\right)^2$
- 变形2：$\frac{2} {\frac{1} {a}+\frac{1} {b}} \leq\sqrt{a b} \leq\frac{a+b} {2} \leq\sqrt{\frac{a^{2}+b^{2}} {2}}$

##### 【微思考】

- $( \frac{a+b} {2} )^{2} \geq a b$ 与 $\frac{a+b}{2} \geq \sqrt{ab}$ 是等价的吗？
- $a + \frac{1}{a} \geq 2$（a $\neq$ 0）是否恒成立？

#### 【典型例题】

##### 题型一：用基本不等式比较大小

**例1** 已知 a, b ∈ R⁺，比较下列各组数的大小：

1. $\frac{a} {b}+\frac{b} {a}$ 和 2
2. $a^2 + 3$ 与 $3a$
3. $a^{2}+\frac{2} {a}$ 与 3

**【变式1】** 设 0 < a < b，则下列不等式中正确的是（ ）

- A. $a<b < \sqrt{ab} < \frac{a+b}{2}$
- B. $a < \sqrt{ab} < \frac{a+b}{2} < b$
- C. $a < \sqrt{ab} < b< \frac{a+b}{2}$
- D. $\sqrt{ab} < a< \frac{a+b}{2} < b$

**【变式2】** 已知 $m = a + \frac{1}{a-2}$（a > 2），$n = 2^{2-b^2}$（b $\neq$ 0），则 m、n 之间的大小关系是（ ）

- A. m > n
- B. m < n
- C. m = n
- D. m ≥ n

##### 题型二：用基本不等式求最值

**例2** 当 x > 0 时，$y = 4x + \frac{12}{x}$ 的最小值为（ ）

- A. 4

**练习：** 已知 a、b、c 为正数，且 a + b + c = 1，证明：$\frac{1}{a} + \frac{1}{b} + \frac{1}{c} \geq 9$

### 三、第10课时：利用基本不等式求最值

#### 【讲授新课】

##### 知识点：应用基本不等式求最值

已知 x、y 都是正数，则：

1. **如果积 xy 等于定值 P**，那么当 x = y 时，和 x + y 有最小值 $2\sqrt{P}$
2. **如果和 x + y 等于定值 S**，那么当 x = y 时，积 xy 有最大值 $\frac{S^2}{4}$

##### 【微思考】

利用基本不等式求最值时应注意哪几个条件？若求和（积）的最值时，一般要确定哪个量为定值？

> **口诀：一正、二定、三相等**

#### 【典型例题】

##### 题型一：利用基本不等式求函数的最值

**例题** 函数 $y = x + \frac{1}{x}$（x > 0）的最小值为____

**变题：** 函数 $y = x + \frac{1}{x}$（x < 0）的最大值为____

1. 函数 $y = x + \frac{1}{x-1}$（x > 1）的最小值为____
2. 函数 $y = \frac{x^2}{x-1}$（x > 1）的最小值为____

**变题：** 函数 $y = \frac{x-1}{x^2}$（x > 1）的最大值为____

3. 函数 $y = x(1-x)$（0 < x < 1）的最大值为____
4. 函数 $y = x(1-2x)$（0 < x < $\frac{1}{2}$）的最大值为____

##### 题型二：等号成立的条件

**例2** 下列命题中为真命题的有（ ）

- A. 函数 $y = \sqrt{x^2+1} + \frac{4}{\sqrt{x^2+1}}$ 的最小值为 4
- B. 不等式 $\sqrt{x^2+1} + \frac{4}{\sqrt{x^2+1}} \geq 4$ 恒成立
- $x^{2}+\frac{1} {x} \geq2 \sqrt{x}$ 对任意 x > 0 恒成立。
- x=1 时，$y=x^{2}+{\frac{1} {x}} ( x > 0 )$ 最小值是 2
