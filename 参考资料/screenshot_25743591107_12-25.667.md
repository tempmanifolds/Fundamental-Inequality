# 基本不等式——基本工具

## 基本不等式是求最值的工具，是生成其它不等式的工具

### 平均数链

$$\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leq \sqrt{ab} \leq \dfrac{a+b}{2} \leq \sqrt{\dfrac{a^2 + b^2}{2}}$$

$$\Downarrow$$

$$\sqrt[n]{\dfrac{a_1^n + a_2^n + \cdots + a_n^n}{n}} \geq \dfrac{a_1 + a_2 + \cdots + a_n}{n} \geq \sqrt[n]{a_1 a_2 \cdots a_n} \geq \dfrac{n}{\dfrac{1}{a_1} + \dfrac{1}{a_2} + \cdots + \dfrac{1}{a_n}}$$

### 推广到 n 元 n 次不等式

由二元二次齐次不等式 $a^2 + b^2 \geq 2ab$ 两边同时加上 $2ab$，得

$$(a + b)^2 \geq 2^2 ab$$

由此也可以类比得到三元三次齐次不等式：

$$(a + b + c)^3 \geq 3^3 abc$$

进而最终得到相对的 $n$ 元 $n$ 次不等式：

$$(a_1 + a_2 + \cdots + a_n)^3 \geq n^n a_1 a_2 \cdots a_n.$$
