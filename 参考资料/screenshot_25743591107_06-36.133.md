# 基本不等式——基本性质

## 基本不等式是不等式的一个基本性质

### 实数比大小的基本事实

$$a - b > 0 \Leftrightarrow a > b$$

$$a - b < 0 \Leftrightarrow a < b$$

$$a - b = 0 \Leftrightarrow a = b$$

### 等式与不等式的性质对照表

| 性质 | 等式 | 不等式 |
| --- | --- | --- |
| 对称性 | $a = b \Leftrightarrow b = a$ | $a > b \Leftrightarrow \underline{b < a}$ |
| 传递性 | $a = b, b = c \Rightarrow a = c$ | $a > b, b > c \Rightarrow \underline{a > c}$ |
| 可加性 | $a = b \Rightarrow a \pm c = b \pm c$ | $a > b \Rightarrow \underline{a + c > b + c}$ |
|  |  | $a > b, c > d \Rightarrow \underline{a + c > b + d}$ |
| 可乘性 | $a = b \Rightarrow ac = bc$ | $a > b, c > 0 \Rightarrow \underline{ac > bc}$ |
|  |  | $a > b > 0, c > d > 0 \Rightarrow \underline{ac > bd}$ |
|  |  | $a > b > 0 \Rightarrow \underline{a^n > b^n} \ (n \in \mathbb{N}, n \geq 2)$ |

### 推导链

$$(a - b)^2 = a^2 - 2ab + b^2 \geq 0$$

$$\Downarrow$$

$$a^2 + b^2 \geq 2ab$$

$$\Downarrow$$

$$\dfrac{a+b}{2} \geq \sqrt{ab}$$
