# 进一步认识基本不等式——函数视角

## 平均数链

$$\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leq \sqrt{ab} \leq \dfrac{a+b}{2} \leq \sqrt{\dfrac{a^2 + b^2}{2}}$$

## 用统一函数刻画

令 $x = \dfrac{a+b}{2}$，则 $x - a = b - x$，从而 $\dfrac{b - x}{x - a} = 1 = \dfrac{a}{a} = \dfrac{b}{b}$。

设 $\dfrac{b - x}{x - a} = \lambda$，则 $x$ 随 $\lambda$ 变化：

- $\lambda = \dfrac{b}{a}$　 $\Rightarrow$　 $x = \dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}}$
- $\lambda = \sqrt{\dfrac{b}{a}}$　 $\Rightarrow$　 $x = \sqrt{ab}$
- $\dfrac{b - x}{x - a} = \dfrac{x + a}{b + x}$　 $\Rightarrow$　 $x = \sqrt{\dfrac{a^2 + b^2}{2}}$

> 【注】通过引入参数 $\lambda = \dfrac{b-x}{x-a}$，可将调和平均、几何平均、算术平均、平方平均统一为同一函数在不同参数下的取值，从而把平均数链理解为 $f(\lambda)$ 的单调性。
